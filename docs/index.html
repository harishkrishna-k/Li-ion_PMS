About This Project

Overview

A comprehensive battery monitoring system for Li-ion batteries, developed as a B.E. final year project. The system monitors voltage, current, and temperature in real-time, with automatic cooling control and remote data storage via Supabase.

Features

  • Real-time voltage, current, and temperature monitoring
  • 16x2 LCD display with status updates
  • Automatic cooling fan control
  • Supabase cloud database integration
  • Portfolio-ready dashboard

Tech Stack

  • Hardware: NodeMCU ESP32, LM35, ACS712, 16x2 LCD
  • Cloud: Supabase PostgreSQL
  • Frontend: HTML, CSS, JavaScript
  • Deployment: GitHub Pages
