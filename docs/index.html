Project Overview

Problem Statement

Electric vehicles require precise battery monitoring to prevent thermal runaway and maximize cell longevity. This project addresses real-time monitoring of voltage, current, and temperature with automatic cooling control.

Solution

A full-stack IoT system using ESP32 for local processing and Supabase for cloud analytics. Features non-blocking firmware, PID thermal control, and professional hardware design.

Key Results

  • 95% voltage accuracy
  • PID-controlled fan reduces overshoot by 60%
  • Deep sleep reduces power by 99%
  • Real-time cloud sync every 5 seconds
