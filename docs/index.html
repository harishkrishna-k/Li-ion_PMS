Task Isolation

FreeRTOS handles Database I/O on Core 0, while Core 1 manages safety-critical sensing and control.

📈

Kalman Filtering

1D Kalman filter replaces simple averaging to eliminate ACS712 Hall-effect noise and ADC jitter.

🐕

Watchdog Security

Hardware WDT ensures the system reboots if the main loop hangs, preventing uncontrolled battery states.

📑

LUT SOC

State of Charge calculated via linear interpolation of a Li-ion discharge Look-Up Table for 3S packs.
